### 1、检索出库单
#### 请求URL:  
```
/query/output
```

#### 请求方式: 
```
POST
```

#### 请求参数

|参数|是否必选|类型|说明|
|:-----|:-------:|:-----|:-----|
|onepeople      |N       |string  |发放人|

|threepeople      |N       |string  |负责人|

```

```
